# Forza Filters

A single-file web app for applying Instagram-style photo filters. No installation, no server, no internet required — just open `index.html` in any browser.

---

## Español

### ¿Qué es?
Aplicación web de filtros de foto al estilo Instagram. Funciona como un único archivo HTML sin dependencias. Cualquier persona puede usarla abriendo el archivo en un navegador.

### Cómo usar
1. Abre `index.html` haciendo doble clic
2. Arrastra una foto al área central, o haz clic para seleccionarla
3. Haz clic en cualquier miniatura de la barra inferior para aplicar un filtro
4. Pulsa **All filters** para ver todos los filtros a la vez
5. Pulsa **Download** para guardar la imagen resultante como PNG
6. Pulsa el icono de papelera (esquina superior derecha de la imagen) para eliminar la foto y subir otra

### Requisitos

## English

### What is it?
An Instagram-style photo filter app. It works as a single HTML file with no dependencies. Anyone can use it by opening the file in a browser.

### How to use
1. Open `index.html` by double-clicking it
2. Drag a photo into the center area, or click to select one
3. Click any thumbnail in the bottom strip to apply a filter
4. Press **All filters** to see every filter at a glance
5. Press **Download** to save the result as a PNG file
6. Press the trash icon (top-right corner of the image) to remove the photo and upload a new one

### Available filters
| Filter | Description |
|--------|-------------|
| **Original** | No filter |
| **8K** | Ultra sharp, micro-contrast, cool tone |
| **Antigua** | Sepia, film grain, heavy vignette |
| **Vivid** | Saturated warm colors, crushed highlights |
| **Japonesa** | Cold desaturated, high contrast, minimalist |
| **Polaroid** | Low contrast, warm whites, green shadows |
| **Noir** | High-contrast black and white |
| **Dreamy** | Pastel, lifted shadows, soft glow |
| **Y2K** | 2-megapixel digital camera look, early 2000s |
| **Golden** | Golden hour, warm skintones, sunset feel |
| **Film** | Kodak Portra simulation, warm and natural |
| **Pacific** | Cool teal-blue tone, VSCO-style, travel |
| **Moody** | Cinematic teal-orange grade, dramatic |
| **Fade** | Lifted blacks, desaturated, editorial style |
| **Bloom** | Bright, airy, rosy, lifestyle look |

### Requirements
- Any modern browser (Chrome, Firefox, Edge, Safari — any version from the last 5 years)
- No installation or internet connection needed

### Anleitung
1. `index.html` per Doppelklick öffnen
2. Ein Foto in den mittleren Bereich ziehen oder klicken, um eines auszuwählen
3. Auf eine Miniaturansicht in der unteren Leiste klicken, um einen Filter anzuwenden
4. **All filters** drücken, um alle Filter auf einen Blick zu sehen
5. **Download** drücken, um das Ergebnis als PNG zu speichern
6. Auf das Papierkorb-Symbol (oben rechts am Bild) drücken, um das Foto zu entfernen und ein neues hochzuladen
