# Forza Filters

A single-file web app for applying photo filters, transforming images, and splitting them into Instagram carousel tiles. No installation, no server — just open `index.html` in any browser.

---

## Español

### ¿Qué es?
Aplicación web de edición de fotos orientada a coches y contenido para redes sociales. Funciona como un único archivo HTML sin dependencias externas. Abre el archivo en un navegador y ya está.

### Interfaz
- **Barra lateral izquierda** — herramientas de transformación y ajuste
- **Área central** — previsualización de la imagen a pantalla completa
- **Barra inferior** — tira de filtros con scroll horizontal (arrastra o desliza)
- **Cabecera** — pestañas de navegación y botón de descarga

### Cómo usar — Filtros
1. Abre `index.html` haciendo doble clic (o despliégala en Vercel/Netlify)
2. Arrastra una foto al área central, o haz clic para seleccionarla
3. Haz clic en cualquier miniatura de la barra inferior para aplicar un filtro
4. Ajusta la **intensidad** del filtro con el slider de la barra lateral
5. Pulsa **All Filters** para ver todos los filtros y presets de un vistazo
6. Pulsa **Download** (cabecera) para guardar la imagen como PNG a resolución original

### Herramientas de edición
| Herramienta | Función |
|-------------|---------|
| **Crop** | Recortar con ratio libre o preestablecido (1:1, 4:3, 16:9, 9:16…) |
| **Rotate L / R** | Rotar 90° a la izquierda o derecha |
| **Flip H / V** | Voltear horizontal o verticalmente |
| **Compare** | Vista antes/después con divisor deslizable |
| **Adjust** | Editor de filtro personalizado (brillo, contraste, saturación, calor, fade, grano, viñeta, nitidez) con opción de guardar como preset |

### Cómo usar — Grid Split
La herramienta Grid Split divide una imagen panorámica en tiles individuales para publicarlos como carrusel de Instagram.

1. Cambia a la pestaña **Grid Split**
2. Arrastra una imagen panorámica al área de previsualización
3. Elige el número de tiles (2, 3, 4 o 6)
4. Elige la proporción de cada tile (9:16, 1:1, 4:3)
5. **Arrastra la imagen** en la previsualización para reposicionar el sujeto dentro del encuadre
6. La barra inferior muestra una previsualización de cada tile individual
7. Pulsa **Download ZIP** para descargar todos los tiles como archivos JPEG numerados

### Requisitos
- Cualquier navegador moderno (Chrome, Firefox, Edge, Safari — versiones de los últimos 5 años)
- Sin instalación ni conexión a internet

## English

### What is it?
A photo editing web app focused on cars and social media content. It works as a single HTML file with no external dependencies. Open the file in a browser and it's ready.

### Interface
- **Left sidebar** — transform and adjustment tools
- **Center area** — full-screen image preview
- **Bottom bar** — filter strip with horizontal scroll (drag or swipe)
- **Header** — navigation tabs and download button

### How to use — Filters
1. Open `index.html` by double-clicking it (or deploy to Vercel/Netlify)
2. Drag a photo into the center area, or click to select one
3. Click any thumbnail in the bottom strip to apply a filter
4. Adjust filter **intensity** with the sidebar slider
5. Press **All Filters** to see every filter and custom preset at a glance
6. Press **Download** (header) to save the image as a full-resolution PNG

### Editing tools
| Tool | Function |
|------|----------|
| **Crop** | Crop with free or preset ratio (1:1, 4:3, 16:9, 9:16…) |
| **Rotate L / R** | Rotate 90° left or right |
| **Flip H / V** | Flip horizontally or vertically |
| **Compare** | Before/after split view with draggable divider |
| **Adjust** | Custom filter editor (brightness, contrast, saturation, warmth, fade, grain, vignette, sharpness) with save-as-preset |

### How to use — Grid Split

3. Choose the tile count (2, 3, 4, or 6)
4. Choose the tile ratio (9:16, 1:1, 4:3)
